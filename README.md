# Pewlett-Hackard-Analysis

## Overview
In this analysis we are helping Pewlett Hackard, a large company with several thousand employees. Many are approaching retirement age and the company needs to be ready to fill the vacancies left behind in their wake. The main questions we will be looking at are, who will be retiring in the next few years, and how many positions will Pewlett Hackard need to fill? As well as who may be able to mentor new hires. We will build a data base using SQL with 6 CSVâ€™s of varying lengths and then use them to filter, join and manipulate in order to find the information we seek.
